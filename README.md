# FloresRegalo 🌸

Un sitio web simple y elegante para mostrar y regalar flores en línea.  
Este proyecto está desarrollado con **HTML, CSS y JavaScript**, y tiene como objetivo ofrecer una experiencia visual atractiva para usuarios que buscan explorar diferentes tipos de flores y sus significados.

---

##  Características
- Catálogo de flores con imágenes y descripciones.
- Diseño responsivo para dispositivos móviles y escritorio.
- Navegación intuitiva y ligera.
- Secciones dedicadas a tipos de flores y sus significados.
- Posibilidad de personalizar mensajes de regalo.

---

##  Estructura del proyecto
- `index.html` → Página principal.
- `FLORES.html` → Catálogo de flores.
- `style.css` → Estilos del sitio.
- `script.js` → Funcionalidades interactivas.
- `assets/` → Carpeta con imágenes y recursos.
